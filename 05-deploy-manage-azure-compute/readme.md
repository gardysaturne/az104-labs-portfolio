# Azure Lab Week 5 (AZ-104): Deploy & Manage Azure Compute Resources

## Lab Theme
**"Global remote-first compute platform"**

This lab set is built around a realistic remote organization with users and IT operations staff across:
- New York
- Tampa
- London
- Sao Paulo
- Johannesburg

The purpose of this week is to strengthen my weakest AZ-104 compute topics by building and validating Azure compute services directly in the Azure Portal.

These labs focus on:
- Azure Container Instances (ACI) container groups
- Azure Container Apps (ACA) vs Azure Kubernetes Service (AKS)
- App Service Plan fundamentals
- App Service Plan isolation decisions
- Scale up vs scale out
- Autoscale behavior
- Deployment slots
- Custom domains
- Backup and restore
- Application Insights

This week is not about random clicking. It is about understanding **what object controls the behavior**, **why a service was chosen**, and **how to prove the configuration in the portal**.
